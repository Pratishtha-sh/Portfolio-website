 # Pratishtha - Personal Portfolio

A modern, responsive, and dynamic personal portfolio built with React and Vite. The site showcases personal projects, work experience, certifications, and skills using sleek animations and interactive 3D elements.

## Features

- **Dynamic Hero Section**: Interactive 3D models and animated components using framer-motion and GSAP.
- **Projects Showcase**: Detailed project templates that render from reusable data logic, complete with automated carousels.
- **Experience Timeline**: A clean chronological display of roles and responsibilities.
- **Responsive Navigation**: Smooth scrolling between sections with an active state highlighter.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & GSAP
- **3D Rendering**: React Three Fiber / Drei

## Setup and Installation

1. **Clone the repository** (if not already local).
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. Open the localhost URL (typically `http://localhost:5173`) in your browser to view the portfolio.

## Building for Production

To create a production-ready build:
```bash
npm run build
```
This will output the optimized static files into the `dist` directory.
